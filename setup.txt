1) Create three files client, backend, and blockchain
2) Move to the blockchain directory and install truffle "npm install -g truffle"
   Truffle is a development environment for Ethereum smart contracts
3) Compile the contract using truffle compile command "truffle compile"
4) After compilation add deployment script in migration folder
5) Run migrate command "truffle migrate" (Make sure the Ganache is open and running)

6) Now initialize your backend "npm init -y"
7) Install required packages "npm install express mongoose dotenv cors"
8) Install nodemon "npm install nodemon --save-dev"
9) Update the scripts in the package.json
   "scripts": {
     "start": "node server.js",
     "dev": "nodemon server.js"
   },
10) Install MongoDB "npm install mongodb"
11) Install web3 for interacting with the Ethereum blockchain
    Install ipfs-http-client for IPFS integration
    "npm install web3 ipfs-http-client"
12) Install multer for file upload functionality "npm install multer"

13) Now lets move to the frontend part. Install vite react "npm create vite@latest ./"
